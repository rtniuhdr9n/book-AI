## 一、通用说明

### 1.1 认证方式

系统使用 **JWT Token** 进行身份认证。

**获取Token：**
- 用户登录：`POST /user/login`
- 管理员登录：`POST /admin/login`

**使用Token：**
在请求头中添加：
```
Authorization: Bearer {token}
```

**Token包含信息：**
```json
{
  "userId": 1,
  "username": "张三",
  "role": 0
}
```

**角色说明：**
| 角色值 | 说明 |
|--------|------|
| 0 | 普通用户 |
| 1 | 管理员 |
| 2 | VIP1 |
| 3 | VIP2 |
| 4 | VIP3 |

### 1.2 统一响应格式

```json
{
  "code": 200,
  "message": "Success",
  "data": {}
}
```

**状态码说明：**
| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权/Token过期 |
| 403 | 权限不足 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

### 1.3 分页参数

列表接口默认支持分页：

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| pageNum | int | 否 | 1 | 页码 |
| pageSize | int | 否 | 10 | 每页数量 |
| keyword | string | 否 | - | 搜索关键词 |

**分页响应格式：**
```json
{
  "code": 200,
  "message": "Success",
  "data": {
    "total": 100,
    "pageNum": 1,
    "pageSize": 10,
    "totalPages": 10,
    "list": []
  }
}
```

## 二、用户模块

### 2.1 用户注册

**接口：** `POST /user/register`

**请求头：**
```
Content-Type: application/json
```

**请求体：**
```json
{
  "username": "zhangsan",
  "password": "123456",
  "phone": "13800138000"
}
```

**参数说明：**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 用户名，3-20字符 |
| password | string | 是 | 密码，6-20字符 |
| phone | string | 是 | 手机号 |

**响应示例：**
```json
{
  "code": 200,
  "message": "Success",
  "data": null
}
```

### 2.2 用户登录

**请求体：**
```json

  "code": 200,

    "id": 1,
```

## 六、订单模块

> 所有接口需要携带 Authorization Token

**接口：** `POST /user/order/create`

**请求体：**
```json
{
  "addressId": 1,
  "remark": "请尽快发货",
  "cartItemIds": [1, 2]
}
```

**响应示例：**
```json
{
  "code": 200,
  "message": "Success",
  "data": {
    "id": 1,
    "orderNo": "202401151230450001",
    "userId": 1,
    "username": "zhangsan",
    "addressId": 1,
    "receiverName": "张三",
    "receiverPhone": "13800138000",
    "fullAddress": "广东省深圳市南山区科技园南路88号",
    "totalAmount": 118.00,
    "discountAmount": 0,
    "payAmount": 118.00,
    "status": 0,
    "statusDesc": "待付款",
    "remark": "请尽快发货",
    "createTime": "2024-01-15T12:30:45",
    "items": [
      {
        "id": 1,
        "bookId": 1,
        "bookTitle": "三体",
        "bookCover": "https://example.com/cover.jpg",
        "quantity": 2,
        "unitPrice": 59.00,
        "totalPrice": 118.00
      }
    ]
  }
}
```

### 6.2 订单支付

**接口：** `POST /user/order/pay`

**请求体：**
```json
{
  "orderId": 1,
  "payMethod": "alipay"
}
```

**支付状态说明：**
- 实际项目中应对接支付平台（支付宝/微信）
- 此处为模拟支付，直接修改订单状态

### 6.3 获取订单列表

**接口：** `GET /user/order/list`

**请求参数：**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| pageNum | int | 否 | 页码 |
| pageSize | int | 否 | 每页数量 |

### 6.4 获取订单详情

**接口：** `GET /user/order/detail/{orderId}`

### 6.5 取消订单

**接口：** `PUT /user/order/cancel/{orderId}`

**说明：** 只能取消待付款订单

### 6.6 确认收货

**接口：** `PUT /user/order/confirm/{orderId}`

**说明：** 只能对已发货订单确认收货
